# SERVITEC PRO V13.29.3 - Supabase, documentos móviles y ejecución profesional

Correcciones principales:

- Sincronización real con Supabase si existen variables en Vercel:
  - `VITE_SUPABASE_URL`
  - `VITE_SUPABASE_KEY` o `VITE_SUPABASE_ANON_KEY`
- Conserva localStorage como respaldo local/offline.
- Botón **Sincronizar Supabase** en Configuración.
- Mejor orden móvil en ejecución: botón **Guardar ejecución** al final del flujo.
- Acta e informe mantienen firmas guardadas del área usuaria y técnico.
- Informe muestra evidencias por actividad cuando se guardan en ejecución.
- Campos y tablas optimizados para celular.
- Versión interna sincronizada a V13.29.3.

> Importante: para que lo guardado en celular aparezca en PC, Supabase debe estar activo y la tabla `app_state` creada.
